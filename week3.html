Homework 03

Due: Wednesday, September 11, 2024, 11:59 PM

Important to Note

Note: You're encouraged to submit your work as you complete each question, but only your final submission will be counted. Ensure all three questions are included in that submission.

Note: Once again, for this submission, some parts will not be autograded, so don't worry about the grade you get from Gradescope for turtle-based questions.

Task

  • (30 pts) Write a function middle that takes three numbers as arguments, and returns whichever of the arguments is numerically in the middle. That is, for example, middle(1,2,3), middle(2,3,1) and middle(14,-10,2) should all return 2.

  • (40 pts) Make a copy of your draw function and rename it to draw_color. This function should have a third parameter, color, representing the color of the figure to be drawn at that particular level. Change your recursive call(s) so that the circles and lines will alternate between red and blue at the different levels. You can assume that we will call the function with either "red" or "blue" as the third argument. Hint: how can you determine the color of the next level of recursion given the color of the current level? Also recall that the command turtle.pencolor can be used to set the color of the turtle. As an example, the call draw_color(3,100,"red") should produce something like the image below.

  • (30 pts) Lucas sequences are similar to Fibonacci numbers, in which:
    L(0) = 0
    L(1) = 1
    L(n) = P*L(n-1) - Q*L(n-2)
    For any given integers P and Q. (Fibonacci numbers correspond to the case of P = 1, Q = -1.) Write a function lucas that takes arguments n, p and q and returns the corresponding Lucas number.
