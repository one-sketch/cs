Homework 02

Due: Friday, September 6, 2024, 11:59 PM

Important to Note

Note: You're encouraged to submit your work as you complete each question, but only your final submission will be counted. Ensure all three questions are included in that submission.

Note: As last week, for this submission, some parts will not be autograded, so don't worry about the grade you get from Gradescope for turtle-based questions.

Task

  • (30 pts) Write a function middle that takes three numbers as arguments, and returns whichever of the arguments is numerically in the middle. That is, for example, middle(1,2,3), middle(2,3,1) and middle(14,-10,2) should all return 2.

  • (30 pts) Write a function draw_shape that has two parameters, shape and size. If shape is "circle" it should draw a circle of radius size. If shape is "triangle", it should draw an equilateral triangle of side length size. If shape is anything else, it should do nothing (but not crash!).

  • (40 pts) Write a function draw_good_rectangle that has two parameters, height and width. It should first check if either height or width are greater than 500 or less than 10, or if the aspect ratio (width/height) is more than 10 or less than 0.1. If any of those are true, it should not draw anything and return False. Otherwise, it should draw a rectangle of the given height and width and return True. You can assume that the turtle will be in its default starting position when the function is called.
