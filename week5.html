Homework 03

Due: Wednesday, September 11, 2024, 11:59 PM

Important to Note

Use: This filewhich contains some code for you to build on. You should implement your solutions to all three questions in this file.

Note: You're encouraged to submit your work as you complete each question, but only your final submission will be counted. Ensure all three questions are included in that submission.

Note: Once again, for this submission, some parts will not be autograded, so don't worry about the grade you get from Gradescope for turtle-based questions.

Task

  • (30 pts) Write a function draw_iter that performs the same functionality as the provided draw function, but using a while loop instead of recursion. Note that this draw function is similar to last week's homework but not the same!

  • (30 pts) Write a function sum_even_squares that has one parameter n and uses a loop to add up squares of all the even numbers that are strictly less than n and return the sum. (So, for example, sum_even_squares(8) should return 56 (the sum of 2^2 + 4^2 + 6^2).) You can assume that n is at least 3.

  • (30 pts) Write a function print_triangle that has one parameter n (you can assume that n is an integer). The function should print one asterisk on the first line, two asterisks on the second line, and so on, up to n asterisks on the n-th line. You can assume that n is at least 1. Note that to print something without Python adding a new line at the end, you can use a command like print(thing_to_be_printed,end="") and to simply cause a new line to be printed, you can use print().
